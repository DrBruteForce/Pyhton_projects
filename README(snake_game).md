# 🐍 Snake Game in Python (Tkinter)

A classic Snake Game built using **Python** and the **Tkinter GUI library**.  
Control the snake, eat food, grow longer, and try to beat your high score — but don’t hit the wall or yourself!

This project is beginner-friendly and great for learning:
- Tkinter graphics for GUI
- Event handling
- Game loops
- Object-oriented programming in Python

---

## 🎮 Demo Features

- ✅ Smooth snake movement
- ✅ Keyboard controls (arrow keys)
- ✅ Score tracking
- ✅ Random food spawning
- ✅ Snake grows after eating food
- ✅ Collision detection (wall + self)
- ✅ Game Over screen
- ✅ Centered window layout
- ✅ Easy to customize settings

---

## 🧰 Tech Stack

- Python 3
- Tkinter (built-in with Python)
- random module

No external packages required.
