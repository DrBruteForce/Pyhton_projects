# 🧮 Python Calculator using Tkinter GUI

A simple GUI-based Calculator built using **Python** and the **Tkinter** library.

This project performs basic arithmetic operations with a clean button-based interface.  
It is beginner-friendly and perfect for understanding GUI development and event handling in Python.

---

## ✨ Features

- ✅ Addition (+)
- ✅ Subtraction (-)
- ✅ Multiplication (*)
- ✅ Division (/)
- ✅ Decimal support (.)
- ✅ Clear button
- ✅ Error handling:
  - Syntax Error
  - Division by Zero (Arithmetic Error)
- ✅ Real-time display update

---

## 🛠️ Technologies Used

- Python 3
- Tkinter (Built-in Python GUI library)

No external libraries required.
